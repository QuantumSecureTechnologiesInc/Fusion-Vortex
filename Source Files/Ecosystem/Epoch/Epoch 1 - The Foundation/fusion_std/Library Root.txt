//! # Fusion Standard Library
//! 
//! The `fusion_std` crate provides the essential runtime utilities for Fusion applications.
//! It is built on top of `fusion_core` and provides:
//! 
//! - **IO**: Async input/output streams (`io`).
//! - **FS**: Secure filesystem access (`fs`).
//! - **Networking**: (Coming in Epoch 2).
//! - **Collections**: (Coming in next step).

pub mod error;
pub mod io;
pub mod fs;

// Prelude for easy import
pub mod prelude {
    pub use crate::error::{StdError, StdResult};
    pub use crate::io::{FusionRead, FusionWrite, Stdout, Stdin};
    pub use crate::fs::{FusionFile, FusionPath};
    
    // Re-export core types so users only need fusion_std
    pub use fusion_core::types::{FusionType, ClassicalType, Tensor, Matrix, Qubit};
}
```

## ✅ Quality check
* **Async-First**: All I/O operations utilize `tokio`'s async traits, aligning with modern high-performance standards (essential for the Runtime Actor system later).
* **Security**: `FusionPath` encapsulates `PathBuf` to enforce sanitization (preventing basic `../../` traversal attacks).
* **Error Handling**: `StdError` wraps `io::Error` and `FusionError`, providing a unified failure domain.
* **Dependencies**: Correctly links to the local `fusion_core` path.
