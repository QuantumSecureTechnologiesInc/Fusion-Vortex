Product Guide: Fusion Unified MonolithComponent ID: F-BIN-MONOClassification: Toolchain ArchitectureAuthor: Lead Architect1. Vision & PurposeThe Unified Monolith is the distribution model for Fusion. It collapses the entire toolchain—compiler, build system, linter, formatter, and runtime—into a single, statically linked binary.2. Technical SpecsReplaces: rustc, cargo, rust-analyzer, clippy.Binary Size: ~50MB (Compressed).Communication: Shared Memory.3. Implementation Detail: Shared Memory CompilationBecause the LSP and the Compiler are the same process, the IDE doesn't need to "save to disk" for the compiler to see changes. The LSP passes the in-memory pointer of the AST directly to the codegen unit.4. Why this existsTo eliminate "Environment Rot." If you have the fusion binary, you have the entire ecosystem. There are no external dependencies to install.
