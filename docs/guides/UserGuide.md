
# User Guide

## Introduction

Fusion allows you to write safe and fast code.

## Syntax

### Functions

```rust
fn add(a: Int, b: Int) : Int {
    return a + b;
}
```text

### Variables

```rust
let x: Int = 10;
```text

### Control Flow

```rust
if x > 5 {
    return x;
} else {
    return 0;
}
```text
