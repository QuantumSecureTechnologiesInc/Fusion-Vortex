# Project Organization

This repository was reorganized to separate source, build outputs, and logs.

- `source_archives/` holds legacy or imported source drops.
- `artifacts/` holds build logs, checks, and misc outputs.
- Active code remains in `src/`, `crates/`, `registry/`, `runtime/`, `toolchain/`, `stdlib/`, `docs/`, and `dist/`.

If you need any item moved back, search under `source_archives/` or `artifacts/`.
