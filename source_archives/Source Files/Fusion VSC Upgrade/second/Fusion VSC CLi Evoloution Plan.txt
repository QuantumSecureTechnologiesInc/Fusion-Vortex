This document provides a production-grade, Rust-first implementation plan to execute Paths 1-4 for Fusion. It is designed as a drop-in evolution of the existing Fusion architecture, prioritizing determinism, auditability, and long-lived infrastructure stability.

SECTION I - FOUNDATIONAL PRINCIPLES
Fusion is treated as infrastructure, not a developer tool. Every design decision enforces:
Ģ Determinism over convenience
Ģ Explicit policy over inferred behaviour
Ģ Crash-only semantics
Ģ Protocol stability
Ģ Zero-trust execution
All systems below are additive and backward compatible unless stated otherwise.

SECTION II - PATH 1: STABILITY & HARDENING (LINUX PHASE)
2.1 Exhaustive Policy Enforcement Audits
Add a mandatory PolicyDecisionPoint (PDP) invoked for every MCP action.
File: crates/policy/src/pdp.rs
pub struct PolicyDecisionPoint {
engine: PolicyEngine,
}
impl PolicyDecisionPoint {
pub fn evaluate(&self, ctx: &ExecutionContext) -> PolicyDecision {
self.engine.evaluate(ctx)
}
}
ExecutionContext includes:
Ģ tool_id
Ģ facet
Ģ capabilities_requested
Ģ trust_score
Ģ workspace_hash
Ģ execution_intent
No execution path bypasses PDP.

2.2 Deterministic Execution Recording & Replay
Introduce an append-only Execution Journal.
File: crates/runtime/src/journal.rs
pub struct ExecutionRecord {
pub execution_id: Uuid,
pub mcp_request: Value,
pub policy_decision: PolicyDecision,
pub tool_fingerprint: String,
pub timestamp: u64,
}
All records are:
Ģ content-addressed (SHA-256)
Ģ fsync'd before execution
Replay Mode:
Ģ disables network
Ģ enforces identical policy
Ģ rehydrates execution state
CLI:
fusion replay <execution_id>

2.3 Crash-Only Runtime Design
Fusion never attempts graceful recovery.
Rules:
Ģ Processes are disposable
Ģ State is externalized
Ģ Restart = restore from journal
Implementation:
Ģ Supervisor process
Ģ Child runtime processes
Ģ Heartbeat + kill-on-timeout
File: crates/runtime/src/supervisor.rs

2.4 Formal Protocol Versioning
Hard-lock protocol versions.
MCP header:
"fusion_mcp": "1.0"
Rules:
Ģ Minor = backward compatible
Ģ Major = explicit opt-in
Runtime refuses mismatched versions.

2.5 Compatibility Test Matrix
Add automated extension compatibility certification.
File: crates/compat/src/matrix.rs
Each extension tested for:
Ģ activation
Ģ command execution
Ģ policy compliance
Ģ determinism
Ģ crash resilience
Output:
Ģ certified / degraded / unsupported
