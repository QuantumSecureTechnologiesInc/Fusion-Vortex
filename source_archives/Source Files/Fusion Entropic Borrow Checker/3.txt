The "Production Gap"
Right now, your engine treats every reference as a "Hot / Mutable" (Stream B) particle. This means:

No Sharing: You cannot have two pointers to the same variable, even if they are just reading.

Code that fails: let a = &x; let b = &x; (This should be valid!)

Eternal Life: Your loans never "die." Once you borrow a variable, it is marked as borrowed forever (until the function ends).

Code that fails: let a = &x; ... (later) ... let b = &mut x; (This fails even if a is never used again).

To make this 100% Production Ready, we need to implement Stream A (Immutable Flow) and Loan Termination.

Here is the code to upgrade your engine from "Paranoid Prototype" to "Usable Compiler."

// src/sema/entropy.rs

    fn detect_collisions(&mut self, instructions: &[Instruction]) {
        for (i, instr) in instructions.iter().enumerate() {
            let state = self.flow_states.get(&i).unwrap();

            match instr {
                // Now handling the 'is_mutable' flag from IR
                Instruction::GetAddress { dest: _, var_name, is_mutable } => {
                    let new_loan_kind = if *is_mutable { LoanKind::Mutable } else { LoanKind::Immutable };

                    for existing_loan in &state.active_loans {
                        // We only care about loans targeting the same variable
                        if existing_loan.target == *var_name {
                            
                            // RULE 1: Existing Mutable Loan blocks EVERYTHING.
                            // (Stream B repels both A and B)
                            if existing_loan.kind == LoanKind::Mutable {
                                self.report_collision(existing_loan, var_name, new_loan_kind, i);
                            } 
                            // RULE 2: Existing Immutable Loan blocks MUTABLE.
                            // (Stream A repels Stream B)
                            else if new_loan_kind == LoanKind::Mutable {
                                self.report_collision(existing_loan, var_name, new_loan_kind, i);
                            }
                            // RULE 3: Immutable + Immutable is SAFE.
                            // (Stream A flows with Stream A) -> NO COLLISION.
                        }
                    }
                }
                _ => {}
            }
        }
    }

    // Helper to clean up the code above
    fn report_collision(&mut self, existing: &EntropicLoan, target: &str, new_kind: LoanKind, new_id: usize) {

            id: new_id,
            target: target.to_string(),
            kind: new_kind,
            origin_index: new_id
