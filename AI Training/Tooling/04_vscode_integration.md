**Dataset Category**: Tooling
**Training Level**: Intermediate
**Last Updated**: December 2025 (v1.0.0) (v1.0.0)

## Overview

Fusion’s VS Code extension connects to the language server and emits real-time diagnostics from parser and semantic analyzer outputs.

## Capabilities

- Diagnostics and error highlighting
- Inline type hints and hover docs
- Code actions for fixes
