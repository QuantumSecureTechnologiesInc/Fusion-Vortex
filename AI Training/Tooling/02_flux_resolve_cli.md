## Overview

The Flux-Resolve CLI orchestrates dependency resolution and cache management.

## Commands

- `fusion flux-resolve` (resolve deps)
- `fusion flux-resolve --engine-mode gpu`
- `fusion flux-resolve --check-updates`
- `fusion flux-resolve tree`

## Tips

- Use GPU mode for large graphs.
- Enable Hive Mind with Redis for team caching.
