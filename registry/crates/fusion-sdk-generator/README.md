# fusion-sdk-generator

Automatic SDK generation tool for Fusion APIs.

## Features

- Multi-language support (Rust, Python, TS)
- API client generation
- Type-safe bindings

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-sdk-generator = "0.1.0"
```text

## Usage

```rust
use fusion_sdk_generator::generate;

fn main() {
    generate();
}
```text
