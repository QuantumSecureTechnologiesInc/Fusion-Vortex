# fusion-trie-search

Efficient Trie data structure implementation for fast string searching.

## Features

- Prefix matching
- Auto-completion support
- Compressed Trie optimization

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-trie-search = "0.1.0"
```text

## Usage

```rust
use fusion_trie_search::Trie;

fn main() {
    let mut trie = Trie::new();
}
```text
