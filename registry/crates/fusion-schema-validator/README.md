# fusion-schema-validator

High-performance data schema validation library.

## Features

- JSON Schema support
- Custom validation rules
- Fast validation engine

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-schema-validator = "0.1.0"
```

## Usage

```rust
use fusion_schema_validator::validate;

fn main() {
    validate();
}
```
