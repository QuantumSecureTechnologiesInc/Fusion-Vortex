# llm-rerope

ReRoPE (Rectified Rotary Positional Embeddings) implementation for extended context windows.

## Features

- Long-context support
- Positional optimization
- Attention scaling

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
llm-rerope = "0.1.0"
```text

## Usage

```rust
use llm_rerope::apply;

fn main() {
    apply();
}
```text
