# Fusion Core Library

The core library for the Fusion programming language, providing fundamental types and traits used across the ecosystem.

## Overview

This crate contains the building blocks for Fusion's runtime and standard library.

## Features

- Basic type definitions
- Core traits and interfaces
- Essential utilities

## Usage

Add to `Cargo.toml`:

```toml
[dependencies]
core = { path = "registry/crates/core" }
```text
