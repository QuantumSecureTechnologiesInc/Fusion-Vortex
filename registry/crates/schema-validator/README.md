# schema-validator

Strict schema validation for complex data structures.

## Features

- Type checking
- Format validation
- Extensible rules

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
schema-validator = "0.1.0"
```text

## Usage

```rust
use schema_validator::validate;

fn main() {
    validate();
}
```text
