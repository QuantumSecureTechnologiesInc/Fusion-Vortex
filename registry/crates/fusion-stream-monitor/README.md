# fusion-stream-monitor

Real-time stream processing and monitoring utility.

## Features

- Stream metrics collection
- Latency tracking
- Throughput monitoring

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-stream-monitor = "0.1.0"
```

## Usage

```rust
use fusion_stream_monitor::monitor;

fn main() {
    monitor();
}
```
