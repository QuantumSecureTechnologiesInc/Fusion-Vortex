# fusion-router-mesh

Distributed service mesh router for Fusion microservices.

## Features

- Service discovery
- Load balancing
- Circuit breaking

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-router-mesh = "0.1.0"
```text

## Usage

```rust
use fusion_router_mesh::hello;

fn main() {
    hello();
}
```text
