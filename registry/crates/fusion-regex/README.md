# fusion-regex

High-performance regex engine for Fusion.

## Features

- Fast regex compilation
- Efficient matching algorithms
- Unicode support

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-regex = "0.2.0"
```text

## Usage

```rust
use fusion_regex::hello;

fn main() {
    hello();
}
```text
