# fusion-safety-monitor

Runtime safety monitoring system for Fusion applications.

## Features

- Real-time invariant checking
- Resource usage monitoring
- Anomaly detection

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-safety-monitor = "0.1.0"
```text

## Usage

```rust
use fusion_safety_monitor::hello;

fn main() {
    hello();
}
```text
