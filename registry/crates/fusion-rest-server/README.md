# fusion-rest-server

A lightweight REST server implementation for Fusion applications.

## Features

- Easy route definition
- Middleware support
- JSON handling

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-rest-server = "0.1.0"
```

## Usage

```rust
use fusion_rest_server::hello;

fn main() {
    hello();
}
```
