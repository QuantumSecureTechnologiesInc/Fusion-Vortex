# fusion-service-router

Dynamic routing layer for Fusion microservices architecture.

## Features

- Path-based routing
- Header-based routing
- Weighted traffic splitting

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-service-router = "0.1.0"
```

## Usage

```rust
use fusion_service_router::route;

fn main() {
    route();
}
```
