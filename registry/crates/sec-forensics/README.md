# sec-forensics

Digital forensics library for post-incident analysis.

## Features

- Memory dump analysis
- Log correlation
- Artifact extraction

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
sec-forensics = "0.1.0"
```

## Usage

```rust
use sec_forensics::analyze;

fn main() {
    analyze();
}
```
