# math-sparse-ops

Optimized operations and kernels for sparse mathematics.

## Features

- SIMD-accelerated sparse ops
- Parallel processing
- Cache-oblivious algorithms

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
math-sparse-ops = "0.1.0"
```text

## Usage

```rust
use math_sparse_ops::spmv;

fn main() {
    spmv();
}
```text
