# rl-algorithms

Reinforcement Learning algorithms collection for Fusion AI.

## Features

- PPO, DQN, A3C implementations
- Environment interfaces
- Training utilities

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
rl-algorithms = "0.1.0"
```

## Usage

```rust
use rl_algorithms::Agent;

fn main() {
    let agent = Agent::new();
}
```
