# fusion-ui

Core UI components and primitives for Fusion frontend applications.

## Features

- Component library
- State management
- Event handling

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
fusion-ui = "0.1.0"
```

## Usage

```rust
use fusion_ui::App;

fn main() {
    App::new().run();
}
```
