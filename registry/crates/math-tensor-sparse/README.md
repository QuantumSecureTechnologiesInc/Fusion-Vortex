# math-tensor-sparse

Sparse tensor implementations for high-dimensional data.

## Features

- N-dimensional sparse tensors
- Tensor contraction
- Decomposition support

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
math-tensor-sparse = "0.1.0"
```text

## Usage

```rust
use math_tensor_sparse::Tensor;

fn main() {
    let t = Tensor::new();
}
```text
