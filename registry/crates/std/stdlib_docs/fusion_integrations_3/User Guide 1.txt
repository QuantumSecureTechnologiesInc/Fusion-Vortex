User Guide: Fusion Ion1. Basic UsageIon infers everything from the file extension.# Compile a Rust file (Optimized for THIS machine by default)
ion compile main.rs

# Compile a C++ kernel
ion compile physics.cpp

# Compile a Python script to optimized bytecode (.pyc)
ion compile model.py
2. Advanced TargetsCross-compiling for specific Supernova nodes.# Target the "Nebula" WASM runtime
ion compile logic.rs --target nebula

# Target a specific NVIDIA GPU architecture
ion compile kernel.cu --target gpu:a100
3. Holo-Linking (Hybrid LTO)To link Rust and C++ into a single high-performance binary with inlining:ion link main.rs lib.cpp --output hybrid_app
4. InspectionSee what Ion is doing under the hood.ion inspect main.rs --ir   # View LLVM IR
ion inspect main.rs --asm  # View Assembly
